What is Css?

  1. Css stand for cascading Style Sheet
  2. Css describe how HTML element are to be displayed on the screen, papper or in ther media
  3. Css is not necessary Css saves a lot of work. It can control the layout of multiple wep pages all at once
  4. External stylesheets are store in CSS files

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements1
  • Internal - by using the style element in head section
  • External - by using a link element to link to an external CSS file
  • The most common way to add CSS, is to keep the styles in external CSS files. However in this tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements. ID, on the other hand ,applies a style to one unique element. ID special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In CSS, selecer are used to target a specific element or range of element on a web page. Once an element on a has been targeted, a style or set of styles can be applied to the element.
